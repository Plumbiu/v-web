# v-web

> 使用 cli 工具在线开发 vue 组件！

## 安装

```bash
npm i -g v-web
```

## 使用

终端输入：

```bash
vw s
```

你可以在网页的右边写代码，按下 `alt + enter` 会自动同步到对应文件！
